MEDICAL PORTAL

This module helps to manage day to day activities of hospital and clinic which can be helpful for any Hospital and medical facilities. It helps to manage Patient details,Doctors details, Patient Appointment, Prescription,Hospital details etc.

Features

Manage Patient Details.

The Patient can enter their details.There are tabs that will show patient details like personal details, diseases etc.

Receptionist can create Appointment.

Receptionist can create appointment with the details like Symptoms,urgency level,appointment date etc.

Generate Patient's Prescriptions.

Generate prescriptions with the doctor's and patient's details.

Generate Report for patient details, appointment, prescriptions,doctor and student.

Generate report for details, appointment, prescriptions,etc

Patients

You can enter all the details of patients.All other tabs will show details about patients like personal details ,diseases etc...


Patients Card Report

Reports for Patients Card can be generated.


Appointments

From this tab user can create an appointment for a patient with basic details like Symptoms,urgency level,appointment date etc..


Here we have feature showing calendar to show appointments on particular dates.


Prescriptions

Prescription is feature where after diagnosis of the patient , you can enter the details of prescripted medicaments, its dosage and other details.


Prescription Report

Reports for Prescription can be generated.


Doctors

this feature allows to add details of doctors like education,specifications, personal details etc.


Doctor Report

Reports for doctors can be generated.


Students

this feature allows to add details of students like education,specifications, personal details etc.


Student Report

Reports for students can be generated.
